# The useSorter hook

We have provided a simple hook for sorting tables. This can be
used for plain React tables, as well as Pocota. It is very easy
to use, and operates on arrays that consists of flat objects
(objects where all the field values are plain strings).

In the example below, we have a datastructure Data
`{data1: string, data2: string, data3: string}`. The hook provides
buttons the user can click on to sort the table data, with the same
names as the fields (in the object SortButton).

```
function SortedTable(props) {
  const data = useFetchData<DataType>()
  const { sorted, SortButton } = useSorter<DataType>(data)

  if (sorted === null) return <Loading />

  return (
	<Table rotate={rotate}>
		<Thead>
			<Tr>
			<Th>
				Header 1
				<SortButton.data1>
			</Th>
			<Th>
				Header 2
				<SortButton.data2>
			</Th>
			<Th>
				Header 3
				<SortButton.data3>
			</Th>
			</Tr>
		<Tbody>
			{sorted.map((data) =>
			(<Tr>
				<Td>{data.data1}</Td>
				<Td>{data.data2}</Td>
				<Td>{data.data3}</Td>
			</Tr>)}
		</Tbody>
	</Table>
```

## Options

useSorter supports 3 options:

```
Options {
    defaultSortBy?: string (field in object)
    arrowUp?: JSX.Element
    arrowDown?: JSX.Element
    arrowUpDown?: JSX.Element
}
```

defaultSortBy is limited to only contain the fields in the given
data structure, if you use this hook with TypeScript!
